# Task-2-Data-Visualization-Storytelling
Power BI dashboard using the Superstore dataset to analyze sales, profit, customers, and products through interactive visualizations.

## Objective

The objective of this project is to create an interactive Power BI dashboard that transforms raw sales data into meaningful business insights using effective data visualization and storytelling techniques.

## Tool Used

* Microsoft Power BI Desktop

## Dataset

* Sample Superstore Dataset

## Dashboard Overview

### Page 1 – Executive Overview

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Monthly Sales Trend
* Sales by Category
* Profit by Category
* Sales by Region
* Interactive slicers (Region, Segment, Ship Mode, Month)

### Page 2 – Customer Insights

* Top 10 Customers by Sales
* Sales vs Profit Analysis
* Sales by Ship Mode
* Profit Margin by Category

## Key Business Insights

* Technology contributes the highest sales.
* The West region generates the highest revenue.
* Standard Class is the most frequently used shipping mode.
* A few customers contribute a significant portion of total sales.
* Sales and profit vary across product categories, helping identify profitable areas.
